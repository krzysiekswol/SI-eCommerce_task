Twoim zadaniem jest "pocięcie" layoutu wg wytycznych, z pliku makieta.psd 
Użyj w projekcie preprocessora CSS (dowolnego np. Less lub Scss) i postaraj się wypozycjonować każdy element przy użyciu flexbox lub css grid.
RWD należy wykonać według własnego uznania, z zachowaniem dobrych praktyk. 
Wsparcie przeglądarki Internet Explorer NIE JEST WYMAGANE :)
Zadanie proszę wykonać bez użycia zewnętrznych bibliotek css/js typu bootstrap i jQuery.
Do "cięcia" pliku PSD można użyć darmowego programu Adobe XD

Skończone zadanie umieść na własnym repozytorium github i prześlij do niego link na adres amatlinski@si-ecommerce.pl w tytule wiadomości wpisz "Zadanie rekrutacyjne Frontend"
